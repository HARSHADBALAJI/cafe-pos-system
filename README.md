# ☕ Java Swing Cafe Management GUI

A 3rd Semester Mini Project implementing a basic desktop Cafe Management System.

## 🚀 Overview
This application provides a simple graphical interface for managing a small coffee shop's daily operations.

### Features Included:
- **Menu Interface:** A basic UI to select available cafe items.
- **Order Calculation:** Computes the total cost of selected items.
- **Basic POS:** Generates a text summary of the customer's bill.

## 🛠️ Tech Stack
- **Language:** Core Java
- **GUI Framework:** Java Swing / AWT

## ⚙️ How to Run
1. Clone this repository to your local machine:
```bash
git clone https://github.com/HARSHADBALAJI/cafe-pos-system.git
```
2. Open the project in a Java IDE (Eclipse, IntelliJ IDEA, or NetBeans).
3. Navigate to the `PROJECT_FILE/SRC` directory.
4. Compile and execute the main Java class file to launch the application interface.
