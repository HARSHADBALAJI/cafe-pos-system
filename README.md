# ☕ Java Swing Cafe Management GUI

Welcome to my 3rd Semester Mini Project! This application is a fully functional **Cafe & Coffee Shop Management System** built natively using **Java Swing** for the graphical user interface.

## 🚀 Features
- **Clean Interface:** An intuitive, user-friendly desktop UI made purely with Java Swing components.
- **Order Management:** Select items from the menu, customize orders, and view order summaries dynamically.
- **Billing System:** Automatically calculates pricing totals and generates bills for customers.
- **Interactive Design:** Custom buttons, panels, and layouts for a seamless coffee shop experience.

## 📸 Screenshots
Look inside the `PROJECT_FILE/SCREENSHOTS` folder to see visual previews of the application in action!

## 🛠️ Tech Stack
- **Language:** Java
- **GUI Framework:** Java Swing, AWT

## ⚙️ How to Run
1. Clone this repository to your local machine:
```bash
git clone https://github.com/HARSHADBALAJI/3RD_SEM_MINI_PROJECT_JAVA_SWING_CAFE_COFFEE_SHOP_GUI.git
```
2. Open the project in your favorite IDE (like Eclipse, IntelliJ IDEA, or NetBeans).
3. Navigate to the `PROJECT_FILE/SRC` directory and run the main application file.
